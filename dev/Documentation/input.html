An interactive in-browser simulation tool.



Imagine that you are opening a small restaurant. Given a set number of customers, do you know the number of chefs, waiters, or busboys you need to comfortably serve everyone? Not such a simple question to answer, especially with outside variables such as customer frequency and meal preparation time, etc. Luckily there is a simple solution; design a simulation. But where to start and which tool to use?


There are many simulation tools on the market available for users to create their own simulation examples. Some are very powerful with a high learning curve for the user; others being easy to use, are specialized for specific types of simulations and do not serve as general purpose software. Depending on how deep your pockets are, the “perfect” simulation tool for you does exist somewhere out in the world.


BLUESIM aims to be different from most of the available tools: free, easy to use, browser based, open source, and ideal for simulation of discrete objects, like restaurant customers, flowing through a system, like a restaurant, over time. This limited scope, along with adequate user documentation, gives BLUESIM the gentle learning curve that so many other simulation tools lack.
